# Smart Home SoC using Verilog (Vivado)

## Description
This project implements a Smart Home Automation System using Verilog HDL on FPGA.

## Features
- Temperature-based fan control
- Light automation using sensor
- Motion detection alarm
- Door control using button

## Tools Used
- Vivado Design Suite
- Verilog HDL

## Files
- smart_home_top.v
- temp_control.v
- light_control.v
- security_fsm.v
- door_control.v
- testbench.v

## Output
Simulation, RTL schematic, and synthesis results verified successfully.
